Nodes and Bonds JS Exercise

Imagine you have a network of 5 users (nodes). You want to connect all these users together, so that each has a connection to everyone in the network. How many connections (bonds) would it take to connect all everyone?

I already have an solution for this problem. Let's use the 5 users problem again. Here is how you would solve it:

(5-1) + (4-1) + (3-1) + (2-1) = 10 bonds.

With this math application, you can also find solutions to the triangular number sequence. (For instance, if a pyramid had 117 rows, how many bricks would it need?)
